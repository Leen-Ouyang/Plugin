<< 游戏模式：
`.game new 桌名` 创建游戏（命名可省略）
`.game over` 销毁本桌游戏
`.game state` 查看本桌状态
`.game master` 登记为GM
`.game set 属性=值` 游戏设置
例: `.game set rule=COC7`
`.game set 属性` 查看设置
`.game call` at玩家
`.game join` 玩家登记
`.game kick 玩家ID` 将玩家踢出游戏
`.game exit` 退出游戏
`.game open 桌号` 将当前窗口加入指定游戏
`.game close` 关闭当前游戏
`.game rou 100` 设置(百面)轮盘骰，详见.help roulette
